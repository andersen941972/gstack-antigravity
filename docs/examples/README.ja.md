# 📚 GStack Antigravity：実録ケーススタディ集

GStack Antigravity を導入することで、AI エージェントがあなたのプロジェクトをどのように強力に支援するかを、実際の実行結果（High-Fidelity 再現）とともに紹介します。

これらの事例は、単なる「動作イメージ」ではなく、**実際の商用プロジェクトで得られた高密度な回答をベース**に構成されています。

## 🔍 主要ワークフローの事例一覧

| ワークフロー | 目的 | ケーススタディ・リンク |
| :--- | :--- | :--- |
| **Learn** | プロジェクトの全体像・技術スタック・文脈の高速抽出 | [LEARN_CASE_STUDY.ja.md](./LEARN_CASE_STUDY.ja.md) |
| **QA** | システムテストの実行・バグ発見・修正・再検証の一貫した自動化 | [QA_CASE_STUDY.ja.md](./QA_CASE_STUDY.ja.md) |
| **Review** | アーキテクチャ、セキュリティ、パフォーマンスの多角的コードレビュー | [REVIEW_CASE_STUDY.ja.md](./REVIEW_CASE_STUDY.ja.md) |
| **Investigate** | スタックトレースの特定、根本原因の深掘り、再発防止策の策定 | [INVESTIGATE_CASE_STUDY.ja.md](./INVESTIGATE_CASE_STUDY.ja.md) |
| **Autoplan** | ビジネス価値と技術的堅牢性を両立させた実装計画のドラフト作成 | [AUTOPLAN_CASE_STUDY.ja.md](./AUTOPLAN_CASE_STUDY.ja.md) |
| **Ship** | 変更内容のサマリ、品質チェック、プロフェッショナルな PR 文面の提案 | [SHIP_CASE_STUDY.ja.md](./SHIP_CASE_STUDY.ja.md) |

## 👨‍💻 この事例集の使い方

各ケーススタディでは、以下の 3 つをセットで掲載しています：
1. **プロンプト**: そのままコピペして使える「指示」の例。
2. **実行結果**: GStack が出力する「生の構造」と「詳細度」。
3. **シニアエンジニアの視点**: なぜその回答が「高品質」と言えるのかの解説。

プロジェクトに本アダプターを追加した後は、これらの事例を参考にしながら、AI エージェントへ指示を出してみてください。
